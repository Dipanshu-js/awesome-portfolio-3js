## Portfolio Features

- **3D Intro Loader** — Three.js canvas with orbit rings, wireframe core, and particle cloud
- **Interactive Hero** — Draggable 3D desktop PC model with OrbitControls
- **Project Cards** — 5 custom Three.js scenes with PBR materials, emissive glow, and additive blending particles
- **Timeline Sections** — Education & Work Experience with company logos
- **Animated Skills** — Ball canvas grid for the full tech stack
- **Contact Globe** — 3D Earth canvas with EmailJS contact form
- **Resume Route** — `/resume` served via Vercel serverless function
- **Custom Cursor** — Magnetic cursor + scroll progress indicator
